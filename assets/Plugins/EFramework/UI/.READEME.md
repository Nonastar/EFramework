## 注册UI

方案一: 使用装饰器

```typescript

@RegisterUI({
    layer: UILayer.Common,
    prefab: "db://assets/resources/MainPanel.prafab",
})
export class xxxPanel extends UIPanelBase {
    
}

```

方案二: 使用函数
```typescript

UIRegistry.register(xxxPanel, {prefab: "db://assets/resources/xxxPanel.prafab", layer: UILayer.Common});
```
